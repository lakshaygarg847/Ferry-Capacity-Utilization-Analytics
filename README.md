# Ferry Capacity Utilization & Operational Efficiency Analytics System

## Overview
This project analyzes Toronto Island Ferry ticket sales and redemption data (2015–2025) to identify capacity utilization patterns, congestion periods, idle capacity, and operational inefficiencies.

## Objectives
- Measure ferry capacity utilization
- Detect congestion-prone intervals
- Identify underutilized operating periods
- Support evidence-based operational planning

## KPIs
- Capacity Utilization Ratio: 0.68%
- Congestion Pressure Index: 19.94%
- Idle Capacity Percentage: 18.06%
- Operational Variability Score: 2.03

## Key Findings
- August is the busiest month.
- January is the least utilized month.
- Summer activity is over 11x winter activity.
- Peak activity occurs around 7 PM.
